
#  MCX — Real-Time Multiplayer 2D Shooter 

Welcome to **MCX**, a lightning-fast real-time **multiplayer 2D shooter** built with **Node.js**, **Express**, and **Socket.IO**!
Fight your friends , dodge bullets , and respawn to keep the action going — all running from a single JavaScript file! Check it out. By the way it's all serverside rendered.
